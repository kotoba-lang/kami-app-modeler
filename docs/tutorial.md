# Kami Modeler quick start

1. Open the public app in a current Chromium, Firefox, or Safari browser.
2. Use **New Cube** to add an object and select it in the scene tree.
3. Enter **Edit Mode**, then choose Vertex, Edge, or Face selection.
4. Select geometry in the viewport. Use `E` to extrude, `I` to inset, `G` to move, and `S` to scale.
5. Use **Save** to persist the project locally. Edits are also crash-safely autosaved after a short idle period; the prior save is retained as a recovery copy.
6. Use **Export Project** for a lossless editable EDN document, or **Export glTF** for interchange.
7. Import the exported file into a fresh tab to verify the handoff before deleting the original.

If an operation fails, the app displays an alert at the top of the viewport. Keep the project export when reporting a reproducible problem.
